# Introduction

[**NLTK**](https://www.nltk.org) is a library for Natural Language Processing (NLP) in Python. It have a lot of pre-trained models for many languages.

# Installation

```bash
pip install nltk
```

## Downloading NLTK Data

```python
import nltk
nltk.download("<DATA>")
```

Example:

```python
import nltk
nltk.download("punkt")
```
